# 🚀 LAS Digital Lending

## Overview

Designed and scaled a unified Loan Against Securities platform covering:

* Shares
* Mutual Funds
* Sovereign Gold Bonds

Transformed a manual lending process into a real-time digital journey.

---

## Customer Journey Flow

Identify Customer → Check Eligibility → Fetch Holdings → Calculate Loan → Pledge → eSign → Activate Limit

---

## Key Problems Solved

* Only ICICI Demat users eligible
* Different flows across asset types
* Manual TAT of 4–5 days
* High drop-offs and live issues
* Separate APIs and rule systems

---

## Solution Delivered

* Single journey for ETB / NTB users
* Centralized APIs across products
* Unified Business Rule Engine (BRE)
* Standardized UX across products
* Dynatrace implementation for tracking

---

## Results

* ₹20M/month → ₹300M/month scale
* 370% revenue growth
* 537% customer expansion
* TAT reduced to Instant
* 28% reduction in failures
